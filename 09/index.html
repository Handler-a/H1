podendo usar o (rel=) para definir o que o navegador vê sendo eles:

  • (rel="_next") próxima página
  • (rel="_prev") página anterior
  • (rel="_author") link para o site do autor
  • (rel="_external") link externo de algo que não faz parte da página
  • (rel="_nofollow") link para algo que não faz parte da página, mas que pagou para ali estar, um site não endossado

Podemos usar o (target=) para controlar o destino de links ancorados

  • (target="_blank") abre em uma janela em braco
  • (target="_self") abre sobrepondo a janela atual
  • (target="_top") desfaz todos os frames e abre uma janela no navegador completo
  • (target="_parent") similar ao (_top) mas referencia a janela mãe
